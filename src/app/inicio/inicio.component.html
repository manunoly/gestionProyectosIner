Bienvenidos al sistema para la postulación de proyectos del INER

El objetivo es crear una herramienta informática para la postulación y sistematización de las propuestas de proyectos para postulación interna, acorde al proceso de postulación interna de proyectos INER.

Esta herramienta debe facilitar tanto el proceso de postulación como la consolidación de las propuestas para generar reportes finales de uso interno o externo.
